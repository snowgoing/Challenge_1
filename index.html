Flying can be fun if done properly. Be sure to always follow proper form and etiquette.

First Steps

  • Find someplace high with a steep drop off
  • Ensure sneakers are laced properly
  • Start running
  • When you reach the edge begin the following order of steps
    1. Jump
    2. Thrust chest with conviction
    3. Consider what you've done
    4. Enjoy it while it lasts
